Setup — what needs env vars

  • Meta sync: set META_ADS_ACCESS_TOKEN on Render to a long-lived Business token with ads_read permission. Then add each ad account above (Meta account IDs start with act_). The sync button pulls the last 30 days per click; the scheduler can run it daily.
  • Google Ads: API needs a developer token (1–3 days to approve) + OAuth refresh token. Until then, use the CSV import — Google Ads has a native report export that matches the schema above.
  • Classifier: lives in app/marketing.py as TOPIC_RULES. Add or reorder keyword patterns as your campaign naming evolves.
